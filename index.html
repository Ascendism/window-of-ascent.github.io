Window of Ascent

Using the tools of today to build the freedom of tomorrow — before the window closes.

Thesis

We are in a narrow, temporary window. Ordinary people can still use modern tools — including AI — to build more independent, resilient, materially grounded lives. That window will not stay open forever: access tightens, terms shift, enclosure hardens.

This project is not generic futurism or tool worship. It is about distinguishing tools from the incentives that govern them, reconnecting digital leverage to physical reality, and turning that clarity into decisions about work, land, autonomy, and creative agency.

> enemy != technology
> enemy = misaligned incentives · enclosure · dependence
> rejecting all tools ≈ worshiping all tools

The book

The main vehicle is a two-edition book on the same spine — two orientations, one argument.

Eco edition

Living systems and place first. Policy, platforms, and models read through soil, water, tenure, and local resilience.

Tech edition

Build-and-run systems first. Land, power, and material constraints on what actually ships and who controls it.

From the narrowing window through tool purity, incentive design, honest objections, and a practical exit ramp — written for readers who want a path, not a posture.

Beyond the page

Public outputs

Research, essays, and media that sharpen the thesis — without turning the project into an endless pipeline for its own sake.

Practical tools

Applied work on land discovery, evaluation, and transition planning — scoped to support real decisions, not fantasy homestead content.

Select components may be released openly as they mature. The manuscript and planning workspace remain private during active development.

What this is not

  • Generic AI commentary or hype
  • A startup pitch deck in prose
  • Prepper cosplay or purity politics
  • Software whose main purpose is building more software
